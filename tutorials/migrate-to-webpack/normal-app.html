A “normal” app

Let’s begin with a simple web app and let’s try to make use of webpack step by step.

add app.coffee

$ ->
    # Model:
    userAgent = window.navigator.userAgent

    # View:
    view =
        render: (ua) ->
            $("#user-agent").text ua

    # Controller (render the model):
    view.render userAgent

add index.html

<!DOCTYPE html>
<html>
    <head>
        <script type="text/javascript" charset="utf-8" src="jquery.min.js"></script>
        <script type="text/javascript" charset="utf-8" src="app.coffee.js"></script>
        <link rel="stylesheet" href="styles.less.css" />
    </head>
    <body>
        <h1>Hello World.</h1>
        <p>Your User-Agent is: <span id="user-agent"></span></p>
    </body>
</html>

add styles.less

body {
    background: #eee;
    p {
        border: 1px solid #333;
        border-radius: 3px;
    }
}

Publishing

For publishing this app, you first need to compile app.coffee to app.coffee.js and style.less to style.less.css. Than you may open the app.

Summary

That’s a really simple app.
