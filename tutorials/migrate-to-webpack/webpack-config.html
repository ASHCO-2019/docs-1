Use a configuration file

We want to put our configuration into a file to we donâ€™t have to type all the command line arguments to compile.

webpack use webpack.config.js as configuration if found in the current directory

add webpack.config.js

module.exports = {
    // the entry point
    entry: "./app.coffee",

    output: {
        // the output path
        path: __dirname,

        // the output filename
        filename: "bundle.js"
    },

    module: {
        loaders: [
            // support for coffeescript files
            { test: /\.coffee$/, loader: "coffee-loader" }
        ]
    }
};
